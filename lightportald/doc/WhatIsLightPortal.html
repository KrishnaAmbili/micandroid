What is Light Portal?
Light Portal is an Ajax and Java based Open Source Portal Server and Social Collaboration Software. Besides AJAX Portal UI Application, Light Portal can also supports other RIAs (Flex, Silverlight etc) and Mobile applications (iPhone, Android, Blackberry etc.) with uniform portal server services API.
Features
  • Ajax and Java based Portal Server.
  • Supports JSR 168 Portlet API programming model.
  • Supports RIAs (Flex/Silverlight etc) and Mobile Apps (iPhone/Android/Blackberry etc) with uniform portal server services API.
  • Built in with social network functions, includes user profile, connections, groups, plus over 50 portlets pre-bundled.
  • Supports Personalized pages for all users, end users can easily add, edit or delete page or sub page.
  • Contains multiple pre-built themes and UI window appearances, also supports customization.
  • Supports multiple mobile browsers, such as iPhone, Opera mini etc.
  • Supports Flex/JSON type portlets.
  • Supports Internationalizaiton (i18n).
  • Supports OpenID authentication.

    Light Portal Architecture
    The following diagram shows the major components of the Light Portal. Each section is described in more detail below.
    HTML/JavaScript/AJAX Portal UI App
    HTML/JavaScript/AJAX Portal UI App is a JavaScript library with a large bunch of view templates. Its responsibility includes rendering portal, portal tabs, portlet window, portlet content and communicate with portal server.
    RIA App
    RIA Apps are such as Flex application or Silverlight Applicaiton etc. Light Portal Server offers uniform server services API using JSON. Curently Light Portal project doesn't include this type App yet, plan to develop it in the future version, but it give developers uniform sever services API to develop their own type RIA right now.
    Mobile App
    Mobile Apps are such as iPhone app, Android app, Blackberry app etc. Light Portal Server offers uniform server services API using JSON. Curently Light Portal project doesn't include this type App yet, plan to develop it in the future version, but it give developers uniform sever services API to develop their own type Mobile Application right now.
    Application Framework
    Application Framework includes 2 parts, one is Portal Server, the other is Portlet container. Developers have full access to the same framework APIs used by the core applications.
  • Portal Server
    • Portal Server is a web based application.
  • Portlet container
    • Portlet container is a JSR 168 Portlet container.
    Applications
    Applications are JSR 168 portlets. Light Portal will ship a set of core applications and social network applications, such as user profile, connections, groups, calendar, email client, blog, forum, search, events, pictures, RSS news reader and others.
